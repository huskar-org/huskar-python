## Install

``pip install huskar-sdk-v2``

With extra dependencies of doc:

``pip install "huskar-sdk-v2[doc]"``

With extra dependencies of tests:

``pip install "huskar-sdk-v2[test]"``

Install all:

``make develop``


## Tests

run
``python setup.py test``
or
``make test``
